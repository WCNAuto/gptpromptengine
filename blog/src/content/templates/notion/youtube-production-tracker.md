## Template structure

### Video Production Database
- **Video Title (Title)**: "How I Built a $10k/Month Side Hustle" — the exact title that appears on YouTube, not a working title
- **Status (Select)**: Concept / Script Draft / Recording / Editing / Thumbnail / Scheduled / Live / Reviewed
- **Publish Date (Date)**: Target go-live date that drives all backward deadline calculations
- **Script Deadline (Date)**: When script must be complete to hit publish date — leave empty and it auto-calculates 7 days before recording
- **Recording Date (Date)**: Actual filming day — triggers editor handoff notification when marked complete
- **Editor (Select)**: Team member names or "Self" — drives the Editor Dashboard filtered view
- **Thumbnail Status (Select)**: Not started / Draft / Client review / Approved / Live
- **Sponsor Segment (Checkbox)**: Check if video includes paid promotion — drives compliance review checklist
- **Target Views 48h (Number)**: Expected views in first 48 hours based on channel average
- **Actual Views 48h (Number)**: Real performance — auto-flags videos underperforming by 30%
- **Revenue Target (Number)**: Expected earnings from ads plus sponsors — tracks against actual AdSense data
- **Video Length (Number)**: Target duration in minutes — drives editing time estimates

## How to use it week by week

1. **Every Sunday evening**: Open the Pipeline view and add next week's recording slots. Fill Video Title, Publish Date, and assign Editor. Status starts at Concept.
2. **Monday morning**: Move to Script Draft status for all videos recording this week. The Script Deadline field auto-populates 7 days before Recording Date.
3. **When script is done**: Change status to Recording. This triggers the Recording Prep checklist and notifies your editor of the upcoming handoff date.
4. **Day after recording**: Update status to Editing and fill actual Recording Date. The template calculates if you're ahead or behind your publish schedule.
5. **When video goes live**: Change status to Live and fill Actual Views 48h exactly 48 hours later. Videos missing this target by 30% appear in the Underperforming filter.
6. **End of each month**: Run the Revenue Report view to compare Target vs Actual earnings across all published videos.

## Frequently Asked Questions

### Why does Script Deadline auto-calculate instead of letting me set it manually?
The formula assumes you need 7 days between script completion and recording for prep, guest outreach, and location booking. If you work faster, edit the formula in the Script Deadline property settings to =Recording Date - 3 days or whatever buffer works for your process.

### How do I handle videos that don't have sponsors but still need compliance review?
Uncheck the Sponsor Segment box but manually change Thumbnail Status to "Client review" if you need legal approval for claims or before/after shots. The compliance checklist appears whenever Thumbnail Status hits any review stage, regardless of sponsor status.

### Can I track multiple editors working on the same video?
The Editor field is single-select to keep the handoff process clean. If you have multiple editors, create separate records for "Video Title - Part 1" and "Video Title - Part 2" with different editors assigned. The Pipeline view groups them visually when you sort by Video Title.
