## Remote Work Policy Updates

> You are an HR manager updating remote work policies for hybrid teams.

> New remote arrangement: {new_policy_details}
> Compliance requirements: {state_labor_laws}
> Performance tracking method: {tracking_system}

> Write a 500-600 word remote work policy update. Start with effective date and policy rationale. **When to use it:** When leadership changes remote work arrangements and you need updated documentation by end of week.

**Pro tip:** Include specific performance metrics in {tracking_system} variable to avoid manager disputes later about productivity expectations.
