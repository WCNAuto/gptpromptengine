## Luxury Property Listings

> You are writing a luxury real estate listing description that attracts high-end buyers.
> 
> Property type: {property_type}
> Price range: {price_range}
> Square footage: {square_feet}
> Key luxury features: {three_premium_features}
> Neighborhood: {neighborhood_name}
> Unique selling point: {standout_feature}
> Target buyer: {buyer_demographic}
> Tone: {sophisticated / warm / exclusive}
> 
> Write a 250-300 word listing description that opens with the most compelling luxury feature. Use elevated language without being pretentious. Include specific details about finishes and amenities. End with a call-to-action for private showings.

**When to use it:** When listing properties over $750K that need to stand out in competitive luxury markets.

**Pro tip:** Lead with the one feature that photos can't fully capture—like smart home integration or custom wine cellars—to hook browsers scrolling through listings.

> Distance to amenities: {minutes_to_town_or_airport}

> Write a 300-350 word description emphasizing the waterfront lifestyle and investment potential. Structure it as: water views and access first, interior luxury second, outdoor living third, location benefits last. Use sensory language that helps buyers visualize the experience.

**When to use it:** Marketing waterfront properties to buyers relocating from other states or seeking vacation homes.

**Pro tip:** Mention specific water activities possible from the property—kayaking, fishing, swimming—rather than generic "water access" to help buyers imagine their lifestyle.

## Starter Homes and Family Properties

> 
> Price point: {exact_listing_price}
> Property type: {starter_home / townhome / condo}
> Bedrooms/bathrooms: {bed_bath_count}
> Move-in condition: {turnkey / minor_updates_needed / fixer_potential}
> Neighborhood benefits: {schools / parks / commute / safety}
> Recent updates: {flooring / paint / appliances / roof}
> Monthly costs: {estimated_mortgage / HOA / taxes}
> Growth potential: {appreciation_trends / neighborhood_development}
> 
> Write a 200-250 word description emphasizing affordability and move-in readiness. Structure as: price and value proposition first, practical benefits second, neighborhood advantages third. Use encouraging language that makes homeownership feel achievable.

**When to use it:** Marketing entry-level properties to first-time buyers who need reassurance about affordability and value.

**Pro tip:** Include approximate monthly payment ranges based on current rates—first-time buyers often think in monthly costs, not total price.

## Investment Properties and Rentals

> You are writing a listing for a cash-flowing rental property targeting real estate investors.
> 
> Property type: {single_family / duplex / small_multifamily}
> Current rental income: {monthly_rent_amount}
> Occupancy status: {tenant_in_place / vacant / month_to_month}
> Cap rate: {estimated_cap_rate}
> Recent improvements: {roof / hvac / flooring / appliances}
> Neighborhood rental demand: {high / stable / growing}
> Future potential: {rent_increase_potential / appreciation_trends}
> Management ease: {low_maintenance / good_tenants / professional_managed}
> 
> Write a 250-300 word investor-focused description leading with financial performance. Structure as: current income and returns, property condition and recent updates, rental market strength, growth potential. Use investor language and include relevant financial metrics.

**When to use it:** Marketing investment properties to serious real estate investors focused on cash flow and returns.

**Pro tip:** Include gross rent multiplier (GRM) and mention any assumable financing—investors calculate quickly and favorable financing can make or break a deal.
